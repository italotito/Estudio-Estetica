Este projeto web foi desenvolvido para um Estúdio de estética e bem-estar. O sistema oferece um catálogo interativo de serviços, agendamento online integrado com calendário e processamento de pagamentos via PIX.

Funcionalidades Principais:;

- Catálogo de Serviços: Visualização detalhada de serviços como Depilação a Cera, Massagem Relaxante, Pedras Quentes e Ventosaterapia.
- Agendamento Online: Seleção de datas e horários disponíveis em um calendário interativo.
- Pagamento via PIX (Banco Inter):
    - Geração automática de QR Code e código "Copia e Cola".
    - Integração direta com a API do Banco Inter (v2).

Administração (Simulada)
- Login: Acesso à área restrita.
- Dashboard: Visualização de agendamentos realizados.

Tecnologias Utilizadas
- Frontend: HTML5, CSS3 (Bootstrap 4, Sass), JavaScript (jQuery), Google Fonts, IcoMoon.
- Backend: Node.js, Express.
- Integrações: API Banco Inter (PIX), Google Charts (QR Code).
